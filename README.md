# portfolio-project
This project is a simple website built as part of a 12-week internship. It is created using HTML, CSS, and JavaScript, and is designed to help me learn the fundamentals of web development and version control using GitHub.
## Website Structure

The website consists of the following pages:

- **Home** – Introduction and overview.
- **About** – Details about my background, skills, and experience.
- **Projects** – A showcase of my projects with descriptions and links.
- **Contact** – Ways to connect with me (email, LinkedIn, GitHub, etc.).
